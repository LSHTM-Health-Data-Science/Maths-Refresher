Question 15: Simplify the function

\[ln \left( \prod_{i=1}^n p^x_i (1-p)^y_i \right)\]

Which of these can it be written as?

  2. \(\sum_i x_i log(p) \times y_i log(1-p)\)

  4. None of the above

  3. \(\sum_i log(-\lambda) + \sum_i x_i log(\lambda) - \sum_i log(x!)\)

  4. None of the above
